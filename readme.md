## 依赖与用法

必备：flask、flask_cors

可选（不需要可以注释掉）：tkinter（浏览文件夹功能）、ortools（更快地计算CUE选择）、ffmpeg（截取音频）

运行`app.py`，打开[127.0.0.1:5050](http://127.0.0.1:5050)即可使用

## features

- 读取并简单解析cue文件
- 汇总曲目列表（可搜索）
- 调用ffmpeg返回对应音频
- 单曲播放、循环播放、单曲循环、随机播放！
- 通过文件名自动在同级目录下寻找专辑封面（暂不支持cue内嵌）
- （ctrl+）点击跳转对应cue文件
- 计算如何最高效获得所有音频（不太好用）
